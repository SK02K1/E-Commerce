## How to run app locally

```
git clone https://github.com/SK02K1/E-Commerce.git
cd E-Commerce
npm install
npm start
```

## Features

- Homepage
- Product listing page
- Filters by
  - Price
  - Category
  - Ratings
- Cart Management
- Wishlist Management
- Authentication
  - Login
  - Logout
  - Signup
- Single Product Page
- Loading and Alerts
- User Profile Page
